Требования для запуска 
	jdk 11+
	веб браузер
	
Порядок запуска
	1. В терминале/консоли перейти в папку проекта
	2. Выполнить 
		для Windows 
			gradlew.bat startDatabase
			gradlew.bat startServer
		
		для Linux
			./gradlew startDatabase
			./gradlew startServer
	3. Запустить браузер и открыть страницу localhost:8089
